# Sideload — ENLIGHTEN.MINT.CAFE v1.0.1

## Fastest path (no cable)
1. Download `enlighten-v1.0.1.apk` to your phone (email/Drive/Drop).
2. Tap it. Allow "Install from unknown sources" when prompted.
3. Launch the app.

## If v1.0.0 is already installed
The signatures match, so `adb install -r enlighten-v1.0.1.apk` will upgrade
in place. If you used a different signing identity earlier, uninstall first:
`adb uninstall cafe.mint.enlighten`.

- **Soundscape (Cosmic Mixer)** → full-width layout on landscape/tablet
